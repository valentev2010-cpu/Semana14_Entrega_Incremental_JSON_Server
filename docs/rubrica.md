# Rubrica de evaluacion - Semana 14

| Criterio | Superior | Alto | Basico | Bajo |
|---|---|---|---|---|
| Funcionalidad demostrable | El sitio ejecuta, guarda datos en JSON Server, actualiza tabla/grafico y funciona sin errores visibles. | Ejecuta y guarda datos, con detalles menores por mejorar. | Ejecuta parcialmente o requiere ayuda para demostrar la funcionalidad. | No logra ejecutar o no demuestra funcionamiento. |
| Validaciones | Valida todos los campos con reglas claras y mensajes comprensibles. | Valida la mayoria de campos. | Valida pocos campos o los mensajes son confusos. | No implementa validaciones funcionales. |
| Integracion tecnica | Integra correctamente Vite, JSON Server, Bootstrap, SweetAlert2 y Chart.js. | Integra la mayoria de herramientas correctamente. | Hay errores de integracion o dependencias sin explicar. | No logra integrar las herramientas. |
| Usabilidad y accesibilidad | La interfaz es clara, responsive, legible y facil de usar. | La interfaz es usable, con detalles menores. | La interfaz funciona pero tiene problemas de claridad o visualizacion. | La interfaz dificulta el uso del sitio. |
| Evidencias | Presenta capturas completas de antes, despues, pruebas, consola y `npm run check`. | Presenta la mayoria de evidencias. | Presenta pocas evidencias. | No presenta evidencias suficientes. |
| Documentacion | Informe, checklist y bitacora estan completos y bien explicados. | Documentacion casi completa. | Documentacion incompleta o superficial. | No documenta el proceso. |
